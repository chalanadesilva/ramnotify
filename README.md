# ramnotify
An app that notifys the user when too much RAM is being used.

CURRENTLY NOT READY, THERE ARE STILL FILES THAT NEED TO BE UPLOADED.


## Reqirements

Requres the python interperter and python libraries psutil, plyer, and pygame.
Install libraries by using `pip3 install -r reqirements.txt`.
Install python at [Python.org](python.org). 

## Installation
download the .zip file and then unzip.

### GNU/LINUX
run `cd ramnotify-main`, and then `chmod +x install_gnu-linux.sh`. After that, run `./install_gnu-linux.sh`.
### MacOS
run `cd ramnotify-main`, and then `chmod +x install_macos.sh`. After that, run `./install_macos`.
### Windows
run `cd ramnotify-main`, and then `install_windows.bat`.
